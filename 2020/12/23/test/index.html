  花了9块钱买了台最便宜的华为云学生机，捣鼓一段时间弄好了lamp服务。apache服务默认80端口，需要在安全组里开放入方向规则才能访问公网ip。

  先编译一个用于反弹shell的java文件，得到class文件后上传到服务器的网站里，再上传一个marshalsec-0.0.3-SNAPSHOT-all.jar并开启RMI服务。

  运行这段指令的时候报错了Listener error
java.net.BindException: Address already in use，百度说是端口被占用。一开始我以为是因为在安全组里开了1234端口就不行，后来发现是因为之前的命令使用ctrl+z退出而不是用ctrl+c终止。
